# 自定义数据扩展

[自定义数据格式](https://github.com/microsoft/vscode-custom-data)允许扩展作者不用编写代码，就能简易扩展VS Code的HTML/CSS语言支持。

自定义数据扩展用到的两个[贡献点](/9.%20参考/2.%20贡献点.md)为：

- `contributes.html.customData`
- `contributes.css.customData`

例如，在扩展`package.json`的这个部分中加入：

```json
{
    "contributes": {
        "html": {
            "customData": ["./html.html-data.json"]
        },
        "css": {
            "customData": ["./css.css-data.json"]
        }
    }
}
```

VS Code将加载定义在两种文件中的HTML/CSS实体，并提供对实体诸如自动补全和悬停信息的语言支持。

您可以在[microsoft/vscode-extension-samples](https://github.com/microsoft/vscode-extension-samples)找到[custom-data-sample](https://github.com/microsoft/vscode-extension-samples/tree/main/custom-data-sample)。
