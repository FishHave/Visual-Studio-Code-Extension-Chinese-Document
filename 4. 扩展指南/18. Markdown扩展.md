# Markdown 扩展

Markdown扩展允许您扩展并增强Visual Studio Code内建Markdown预览。这包括更换预览外观或添加对新Markdown语法的支持。

## 使用CSS更改Markdown预览的外观

扩展可以贡献CSS，更改Markdown预览的外观或布局。使用扩展`package.json`中的`markdown.previewStyles`[贡献点](/9.%20参考/2.%20贡献点.md)注册样式表：

```json
"contributes": {
    "markdown.previewStyles": [
        "./style.css"
    ]
}
```

`"markdown.previewStyles"`是一系列文件的对扩展根目录的相对路径。

贡献的样式在内建Markdown预览样式之后添加，但在用户`"markdown.styles"`之前。

[Markdown预览GitHub样式](https://marketplace.visualstudio.com/items?itemName=bierner.markdown-preview-github-styles)是不错的案例，它展示了如何使用样式表让Markdown预览看起来像GitHub渲染的Markdown。您可以在[GitHub](https://github.com/mjbvz/vscode-github-markdown-preview-style)审查扩展源代码。

## 使用markdown-it插件添加对新语法的支持
VS Code Markdown预览支持[CommonMark规范](https://spec.commonmark.org/)。通过贡献[markdown-it插件](https://github.com/markdown-it/markdown-it#syntax-extensions)，扩展可以支持额外Markdown语法。

贡献markdown-it插件，先在扩展`package.json`中贡献`"markdown.markdownItPlugins"`：

```json
"contributes": {
    "markdown.markdownItPlugins": true
}
```

随后，在扩展的主`activation`函数中，返回带有`extendMarkdownIt`函数的对象。这个函数收集当前的markdown-it实例，必须返回新的markdown-it实例：

```typescript
import * as vscode from 'vscode';

export function activate(context: vscode.ExtensionContext) {
    return {
        extendMarkdownIt(md: any) {
            return md.use(require('markdown-it-emoji'));
        }
    };
}
```

如要贡献多个markdown-it插件，将多个`use`链接在一起，并返回：

```typescript
return md.use(require('markdown-it-emoji')).use(require('markdown-it-hashtag'));
```

贡献markdown-it插件的扩展为惰性启用，仅在Markdown预览第一次显示时启用。

[markdown-emoji](https://marketplace.visualstudio.com/items?itemName=bierner.markdown-emoji)展示了使用markdown-it插件为markdown添加emoji支持。您可以在[GitHub](https://github.com/mjbvz/vscode-markdown-emoji)上审阅Emoji扩展的源代码。

您也许想要看看：
- markdown-it插件开发者的[指南](https://github.com/markdown-it/markdown-it/blob/master/docs/development.md)
- [已有的markdown-it插件](https://www.npmjs.com/browse/keyword/markdown-it-plugin)

## 使用脚本添加进阶功能

若追求进阶功能，扩展可能要贡献脚本，在Markdown预览中执行。

```json
"contributes": {
    "markdown.previewScripts": [
        "./main.js"
    ]
}
```

贡献的脚本会异步加载，并在每次内容更改时重新加载。

[Markdown预览Mermaid支持](https://marketplace.visualstudio.com/items?itemName=bierner.markdown-mermaid)展示了使用脚本为Markdown预览添加[Mermaid](https://mermaid.js.org/)图示和流程图支持。您可以在[GitHub](https://github.com/mjbvz/vscode-markdown-mermaid)找到Mermaid扩展的域代码。
