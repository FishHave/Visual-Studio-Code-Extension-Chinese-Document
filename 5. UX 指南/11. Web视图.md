# Web视图

如果您需要显示VS Code API支持外的自定义功能，可以使用完全自定义的[Web视图](/4.%20扩展指南/8.%20Web视图.md)。理解Web视图仅在绝对必要时使用十分重要。

✔️ **要**
- 只在绝对必要时使用Web视图
- 只在上下文上合理时启用扩展
- 只在活跃窗口打开Web扩展
- 确保视图中所有元素可主题化（参考[Web视图示例](https://github.com/microsoft/vscode-extension-samples/blob/main/webview-view-sample/media/main.css)和[颜色词元](/9.%20参考/7.%20主题颜色.md)文档）
- 确保您的视图遵循[易用性指南](https://code.visualstudio.com/docs/configure/accessibility/accessibility)（颜色对比，ARIA标签，键盘导航）
- 在工具栏和视图中使用命令操作

❌ **不要**
- 用于推广（升级、赞助等等）
- 用于引导
- 在每一个窗口打开
- 在扩展升级时打开（使用通知询问替代）
- 添加与编辑器或工作空间无关的功能
- 重复实现已有的操作（欢迎页面、设置、配置等等）

## Web视图

您可以将Web视图放入任何视图容器（边栏或面板），这些元素叫做[Webview视图](/9.%20参考/1.%20VS%20Code%20API.md)，适用Web视图指南。

![](https://code.visualstudio.com/assets/api/ux-guidelines/examples/webview-view.png)

*此Webview视图使用下拉框、输入框和按钮显示了创建拉取请求的内容。*
